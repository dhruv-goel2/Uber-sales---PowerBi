# Uber-sales---PowerBi
🚖 Uber Trip Analysis Dashboard
📌 Project Overview

The Uber Trip Analysis Dashboard is an interactive Business Intelligence project developed in Power BI to analyze Uber trip data and uncover meaningful business insights. The dashboard helps monitor key performance indicators (KPIs), booking trends, revenue, cancellations, vehicle performance, and customer behavior through interactive visualizations.

This project demonstrates how raw operational data can be transformed into actionable insights to support data-driven decision-making.

🎯 Objectives
Analyze overall booking performance.
Monitor revenue and trip trends.
Identify peak booking periods.
Evaluate vehicle-wise performance.
Analyze booking cancellations.
Track completed and cancelled trips.
Create an interactive dashboard for business users.
